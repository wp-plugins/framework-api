== Description ==

Defines an extensible framework for use by other plugins, by keeping the generic code in one place will allow for easier maintenance across a large set of complex plugins.

== Installation ==
1. Verify that you have PHP5, which is required for this plugin.
2. Upload the whole `framework` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
